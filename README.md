## Retina补丁使用
``` bash
ioreg -lw0 | grep IODisplayEDID | sed "/[^<]*</s///" | xxd -p -r | strings -6
B140HAN01.3 
```
查询您的显示屏型号是否符合本补丁，如果您的屏幕与我的屏幕VID不一致请自行修改VID和PID。
``` bash
/系统/资源库/Displays/Contents/Resources/Overrides/
```
修改完成后放入到此目录下并进行重启，可以达到MacBook系列原生开启Retina目的
